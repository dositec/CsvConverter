# CsvConverter

CsvConverter — это простой и гибкий инструмент для конвертации CSV-файлов в Excel (XLSX) с поддержкой настройки колонок через YAML-конфигурацию. Приложение построено на .NET 8 и имеет графический интерфейс Windows Forms.

## Возможности

- Конвертация CSV в Excel (XLSX).
- Настройка привязки столбцов через `Config.yaml`.
- Поддержка новых столбцов без `Input` с фиксированными значениями.
- Автоматическое заполнение пропущенных значений через `DefaultValue`.
- Создание строк с несколькими значениями через `DefaultValues`.
- Группировка значений по колонке `OutputGroup`.
- Преобразование чисел с помощью `Math`.
- Округление результата вверх с помощью `MathRound`.
- Поддержка заголовка отчета, фона заголовков и зебро-стиля строк.
- Логирование ошибок и предупреждений.
- Отслеживание прогресса и возможность отмены конвертации.

## Конфигурация YAML

Конфигурационный файл `Config.yaml` определяет, как поля из CSV должны попадать в Excel.

Пример:

```yaml
Caption: "Заголовок отчёта"
CaptionBackgroundColor: "#4472C4"
HeaderBackgroundColor: "#D9E1F2"
ZebraColors:
  - "#F2F2F2"
  - "#FFFFFF"
Columns:
  - Input: "некоторая задача"
    OutputIndex: 1
    Output: "Задача"
    DefaultValue: "DEFAULT"
  - Input: "номер задачи"
    OutputIndex: 2
    Output: "Номер задачи"
    DefaultValue: "-"
  - Input: "некоторый текст"
    OutputIndex: 3
    Output: "Текст"
    DefaultValue: "Empty"
  - Input: "Имя"
    OutputIndex: 4
    OutputGroup: "Имя"
  - OutputIndex: 5
    Output: "Описание"
    DefaultValues:
      - "А1:"
      - "Б1:"
  - Input: "Время"
    OutputIndex: 6
    Output: "Время"
    Math: "/ 2"
  - OutputIndex: 8
    Output: "Новая Колонка2"
    DefaultValue: "--"
```

### Параметры колонки

- `Input` — имя столбца в CSV. Если параметр не указан, колонка считается новой и заполняется фиксированным значением.
- `OutputIndex` — номер столбца в итоговой Excel-таблице.
- `Output` — название заголовка в Excel.
- `OutputGroup` — указывает колонку для группировки. Значения в этой колонке будут объединяться по группам.
- `DefaultValue` — значение, которое будет записано в колонку, если исходные данные отсутствуют.
- `DefaultValues` — список значений для расширения строк: каждая исходная строка дублируется для каждого значения из списка.
- `Math` — математическая операция над значением из CSV. Пример: `/ 2`, `+ 100`, `* 1.5`.
- `MathRound` — та же математическая операция, что и `Math`, но результат после вычисления округляется вверх (`Math.Ceiling`). Пример: `4 / 100` вернёт `1`.

### Примеры использования

#### Фиксированная колонка без `Input`

```yaml
- OutputIndex: 8
  Output: "Новая Колонка2"
  DefaultValue: "--"
```

Эта колонка всегда будет содержать значение `--`.

#### Колонка с несколькими значениями

```yaml
- OutputIndex: 5
  Output: "Описание"
  DefaultValues:
    - "А1:"
    - "Б1:"
```

Каждая строка из CSV будет развернута на две строки с разными значениями в колонке `Описание`.

#### Математическое вычисление с округлением вверх

```yaml
- Input: "Время"
  OutputIndex: 6
  Output: "Время"
  MathRound: "/ 2"
```

Если значение равно `4`, то результат будет округлён вверх до `1`.

## Работа с приложением

1. Запустите `CsvConverter.exe`.
2. Выберите конфигурационный файл `Config.yaml` через меню.
3. Добавьте один или несколько CSV-файлов.
4. Нажмите **Convert**.
5. Дождитесь завершения и проверьте лог.
